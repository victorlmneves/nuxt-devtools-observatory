## What this module gives you

Nuxt DevTools Observatory adds five focused observability panels to Nuxt DevTools:

- useFetch Dashboard
- provide/inject Graph
- Composable Tracker
- Render Heatmap
- Transition Tracker

Use this documentation for setup, feature walkthroughs, API references, and troubleshooting.

- [Quick Start](/getting-started/installation) - Install the module and verify your first signal.
- [Feature Guides](/feature-guides) - Learn each tab and what to look for.
- [API Reference](/api-reference) - Module options and RPC snapshot contracts.
